Endangered Species of Ontario (ESOO)

ESOO is a site constructed with React that shows basic data surrounding species that are endangered in Ontario. The user can select which category of endangered species they would like to view, after choosing, the program will display the names of the appropriate species relating to the category of species selected. Should the user then click on one of the species names, the app then fetches and displays the scientific name, location, endangered status, date added to the endangered species list in Ontario of the species, and a picture of the animal. The application was built primarily using JavaScript, Node, and React and utilizes a REST and GraphQL API that I developed.

Concisify

During the 2021 Hack the North Hackathon, as a part of a group I helped contribute to the creation of a web application called "Concisify". "Concisify" is a study tool that takes in a text input, such as an article, and returns various fill in the blank type questions related to the content of the article and created by AI. This is done to help test your knowledge of the subject and prepare for any sort of upcoming assessment. The project was built through the use of React, a Flask API, and various Python libraries such as Gensim and NLTK.
